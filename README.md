\- GitHub

\## Objetivo do Software

O sistema tem como objetivo realizar o controle e gerenciamento de visitas em uma organização. A aplicação permite o cadastro de áreas, processos, usuários e visitas, utilizando a linguagem Java integrada a um banco de dados MySQL, possibilitando a visualização de uma agenda organizada das visitas registradas.



\## Funcionalidades do Sistema

\- Cadastro de usuários

\- Autenticação de acesso ao sistema

\- Cadastro de áreas

\- Cadastro de processos

\- Registro de visitas

\- Visualização da agenda de visitas

\- Listagem de visitas em tabela

\- Exclusão de visitas registradas
